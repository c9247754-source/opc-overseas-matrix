# StageListing — 用户要什么

## 用户要什么

房产经纪人要的是：**空房照片 → 能上 listing 的布景图**，比传统 virtual staging 便宜、快。

不是「通用室内灵感」或「又一个去背」。

## MVP

上传空房 → 选风格 → 出图（免费水印 / 付费干净导出）→ Creem Credits $9 / Pro $19

真人像级 staging：配置 `FAL_KEY` 或 `REPLICATE_API_TOKEN`。无 key 时仍有本地 preview，保证链路可上线。
